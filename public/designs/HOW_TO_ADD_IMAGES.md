# How to Add Your Design Images

## üìÅ Adding Images to the Portfolio

1. **Place your design files** in the `/public/designs/` folder
   
2. **Supported formats:**
   - `.jpg` / `.jpeg`
   - `.png`
   - `.webp`

3. **File naming tips:**
   - Use descriptive names: `brand-identity-logo.png`
   - Avoid spaces: use hyphens or underscores
   - Examples:
     - `social-media-campaign.jpg`
     - `event-poster-2024.png`
     - `logo-design-tech-startup.jpg`

4. **The website will automatically:**
   - Detect all images in the folder
   - Display them in the Graphic Design section
   - Apply vintage photo effects
   - Create a masonry grid layout

## üé® Image Recommendations

- **Resolution:** At least 800x800px for quality
- **Aspect Ratio:** Any ratio works (the masonry grid adapts)
- **File Size:** Keep under 2MB for faster loading
- **Format:** PNG for logos/graphics, JPEG for photos

## üîÑ Updating Images

Simply:
1. Add new images to `/public/designs/`
2. Refresh your browser
3. New images appear automatically!

No code changes needed - the portfolio dynamically loads all images from this folder.
